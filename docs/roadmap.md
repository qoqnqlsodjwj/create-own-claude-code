## 学习建议

### 时间分配
- 每天投入 **1-2 小时**
- 80% 写代码，20% 读文档
- 每个模块先跑通代码，再理解原理

### 学习方法
1. **先运行** — 每步代码都可以独立运行
2. **再修改** — 改参数看效果
3. **然后理解** — 读注释和 README
4. **最后挑战** — 完成 exercises.md 中的练习

### 优先级削减
如果时间不够，按这个顺序砍：
1. 模块 4 (终端 UI) — 可以用简单 console.log 替代
2. 模块 6 (Agent 系统) — 单 Agent 也能用
3. 模块 7 (MCP) — 内置工具足够
4. 其他模块尽量不砍
